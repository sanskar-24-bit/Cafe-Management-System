📌 Project Overview 

A beginner-level Python console application that simulates a basic restaurant ordering system for SHINDE Restaurant. It allows a customer to browse the menu, place up to two orders, 
and get the total bill.

🛠️ Technologies Used
Language: Python 3
IDE: VS Code

🍽️ Menu
Item   Price
Pizza  Rs. 40
Pasta  Rs. 50
Burger Rs. 60
Salad  Rs. 70
Coffee Rs. 80

⚙️ Features
Displays a welcome message and full menu on launch
Accepts the customer's first item order
Validates whether the item exists in the menu
Prompts to add a second item (Yes/No)
Calculates and displays the total bill amount
